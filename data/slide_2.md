# What traefik is ...

Traefik is a modern HTTP reverse *proxy* and *load balancer* that makes deploying microservices easy. 
Traefik configures itself automatically and dynamically.


# System Integration
* Docker, 
* Swarm mode, 
* Kubernetes, 
* Amazon ECS
* ...
